**3. _Subarray ရှာ ခြင်းပုဒ်စာ_**
[2025A_FE-B_Q3.cpp](2025A_FE-B_Q3.cpp)
<details>
  
ဒီပုစ္ဆာက Array တစ်ခုထဲမှာရှိတဲ့ ကပ်လျက်ကိန်းဂဏန်းများပေါင်းလဒ် _(Subarray Sum)_ ဟာ ကိုယ်ရှာချင်တဲ့ targetSum နဲ့ တူသလားဆိုတာကို <br/>
Nested Loop သုံးပြီး ရှာခိုင်းတာဖြစ်ပါတယ်။

**၁။ ကွက်လပ် A ကို စဉ်းစားခြင်း (Loop ပတ်ရမည့် အဆုံးသတ်)**
<br/><br/>
အတွင်းပတ်လမ်း (Inner loop) ဖြစ်တဲ့ end variable က subarray ရဲ့ အဆုံးသတ် index ကို ကိုယ်စားပြုပါတယ်။ <br/>
Subarray တစ်ခုဟာ array ရဲ့ နောက်ဆုံးကိန်းအထိ ဖြစ်နိုင်တဲ့အတွက် end loop ကို array ရဲ့ အဆုံးသတ်ဖြစ်တဲ့ N အထိ ပတ်ပေးရပါမယ်။ <br/><br/>

ဒါ့ကြောင့် A = N ဖြစ်ပါတယ်။

**၂။ ကွက်လပ် B ကို စဉ်းစားခြင်း (ပေါင်းလဒ်ကို တွက်ချက်ခြင်း)**
<br/><br/>

အတွင်းပတ်လမ်းထဲမှာ ကိန်းဂဏန်းတွေကို တစ်ခုပြီးတစ်ခု ပေါင်းထည့်ရမှာပါ။ <br/>
sum ထဲကို လက်ရှိရောက်နေတဲ့ end index မှာရှိတဲ့ တန်ဖိုးကို အရင်ရှိပြီးသား sum နဲ့ ပေါင်းရပါမယ်။<br/><br/>

ဒါ့ကြောင့် B=sum + arr[end] ဖြစ်ပါတယ်။

**5. _သုဒ္ဓကိန်း (Prime Numbers) ရှာခြင်း_**
[2025A_FE-B_Q5.cpp](2025A_FE-B_Q5.cpp)

<details>
  
**၁။ ကွက်လပ် A ကို စဉ်းစားခြင်း (Prime ဟုတ်၊ မဟုတ် စစ်ဆေးခြင်း)**
<br/><br/>
သုဒ္ဓကိန်းဆိုတာ ၁ နဲ့ ၎င်းကိန်းကိုယ်တိုင်ကလွဲရင် တခြားကိန်းနဲ့ စားမပြတ်တဲ့ ကိန်းမျိုးပါ။

for loop ထဲမှာ number mod i = 0 (စားပြတ်သွားတယ်) ဆိုရင် အဲဒီကိန်းဟာ သုဒ္ဓကိန်း မဟုတ်တော့ပါဘူး။

ဒါကြောင့် isPrime ဆိုတဲ့ variable ကို false လို့ ပြောင်းပေးရပါမယ်။

A = isPrime ← false ဖြစ်ပါတယ်။

**၂။ ကွက်လပ် B ကို စဉ်းစားခြင်း (အရေအတွက်ကို မှတ်သားခြင်း)**
<br/><br/>
အပြင်က while loop က count ≤ N အထိ ပတ်နေတာပါ။ <br/>
သုဒ္ဓကိန်း တစ်လုံးကို ရှာတွေ့ပြီး output ထုတ်လိုက်တိုင်းမှာ count (ရှာတွေ့ပြီးသမျှ အရေအတွက်) ကို တစ်တိုးပေးရပါမယ်။

သုဒ္ဓကိန်းဖြစ်မှသာ (if isPrime = true) output ထုတ်ပြီး count ကို တစ်တိုးရမှာ ဖြစ်ပါတယ်။

B = count ← count + 1 ဖြစ်ပါတယ်။
