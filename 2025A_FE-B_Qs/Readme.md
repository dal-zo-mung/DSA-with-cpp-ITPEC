**2. _mode အများဆုံး ပါ ဝင်တဲ့ number ကိန်း ကို ရှာ ခြင်း_**

**မေးခွန်း၏ အဓိပ္ပာယ်**
  
  Dataset (ကိန်းဂဏန်းအစုအဝေး) တစ်ခုထဲမှာ အကြိမ်အရေအတွက် အများဆုံးပါဝင်တဲ့ တန်ဖိုးကို Mode လို့ ခေါ်ပါတယ်။ <br/>
  ဥပမာ - {2, 1, 1, 9, 6, 6, 2, 5, 6} ဆိုတဲ့ အစုထဲမှာ 6 က သုံးကြိမ်တောင် ပါတဲ့အတွက် သူ့ကို Mode လို့ သတ်မှတ်ပါတယ်။ <br/>
  ဒီ ပုဒ်လေးကတော့ Array ထဲက ကိန်းဂဏန်းတွေကို တစ်ခုချင်းစီ စစ်ဆေးပြီး အကြိမ်ရေ အများဆုံးပါတဲ့ ကိန်းကို ရှာဖွေပေးရတာ ဖြစ်ပါတယ်။
