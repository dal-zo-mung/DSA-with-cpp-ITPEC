## DSA-with-cpp-ITPEC

ITPEC (FE) စာမေးပွဲအတွက် အရေးအကြီးဆုံးဖြစ်တဲ့ အောက်ပါ Logic (၇) မျိုးကို C++ နဲ့ အဓိကထားပြီး Implement လုပ်ထားပါတယ်။

---

### 🔍 Searching Algorithms (ရှာဖွေခြင်း)
အချက်အလက်တွေထဲက ကိုယ်လိုချင်တဲ့ Data ကို ထိရောက်စွာ ရှာဖွေနိုင်တဲ့ နည်းလမ်း (၂) မျိုး-

1. Linear Search: ဒေတာတွေကို တစ်ခုချင်းစီ အစဉ်လိုက် လိုက်ရှာတဲ့ အခြေခံ logic။

2. Binary Search ⭐️: Sort လုပ်ပြီးသား ဒေတာတွေကို တစ်ဝက်စီ ခြမ်းထုတ်ပြီး အချိန်တိုအတွင်း ရှာဖွေနိုင်တဲ့ အဆင့်မြင့် logic။

---

### 🔄 Sorting Algorithms (အစဉ်လိုက် စီစဉ်ခြင်း)
ဒေတာတွေကို အကြီးအသေး အစီအစဉ်အလိုက် စီတဲ့အခါ သုံးတဲ့ နည်းလမ်း (၆) မျိုး-

1. Bubble Sort: အနားချင်းကပ်လျက် ဒေတာတွေကို နှိုင်းယှဉ်ပြီး လဲလှယ်သွားတဲ့ logic

2. Selection Sort: အသေးဆုံးဒေတာကို ရှာပြီး အရှေ့ကို ပို့ပေးတဲ့ နည်းလမ်း။

3. Insertion Sort: ဒေတာအသစ်ကို သင့်တော်တဲ့နေရာထဲသို့ ကြားညှပ်ထည့်သွင်းတဲ့ logic။

4. Shell Sort: Insertion sort ကို ပိုမိုမြန်ဆန်အောင် Gap (ကြားအကွာအဝေး) ခြားပြီး စီတဲ့နည်းလမ်း။

5. Quick Sort: Pivot တစ်ခုထားပြီး အုပ်စုခွဲကာ အမြန်ဆုံး စီနိုင်တဲ့ logic။

6. Merge Sort: Divide and Conquer နည်းစနစ်နဲ့ ဒေတာတွေကို ခွဲထုတ်ပြီး ပြန်ပေါင်းကာ စီတဲ့ logic။

---
