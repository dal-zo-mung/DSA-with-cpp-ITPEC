# DSA-with-cpp-ITPEC

ITPEC (FE) စာမေးပွဲအတွက် အရေးအကြီးဆုံးဖြစ်တဲ့ အောက်ပါ Logic (၇) မျိုးကို C++ နဲ့ အဓိကထားပြီး Implement လုပ်ထားပါတယ်။ အထူးသဖြင့် Star (⭐️) ပြထားတဲ့ Algorithm တွေကို ပိုမို အလေးပေး ရေးသားထားပါတယ်။

---

### 🔍 Searching Algorithms (ရှာဖွေခြင်း)
အချက်အလက်တွေထဲက ကိုယ်လိုချင်တဲ့ Data ကို ထိရောက်စွာ ရှာဖွေနိုင်တဲ့ နည်းလမ်း (၂) မျိုး-

Linear Search: ဒေတာတွေကို တစ်ခုချင်းစီ အစဉ်လိုက် လိုက်ရှာတဲ့ အခြေခံ logic။

Binary Search ⭐️: Sort လုပ်ပြီးသား ဒေတာတွေကို တစ်ဝက်စီ ခြမ်းထုတ်ပြီး အချိန်တိုအတွင်း ရှာဖွေနိုင်တဲ့ အဆင့်မြင့် logic။

---

### 🔄 Sorting Algorithms (အစဉ်လိုက် စီစဉ်ခြင်း)
ဒေတာတွေကို အကြီးအသေး အစီအစဉ်အလိုက် စီတဲ့အခါ သုံးတဲ့ နည်းလမ်း (၅) မျိုး-

Bubble Sort: အနားချင်းကပ်လျက် ဒေတာတွေကို နှိုင်းယှဉ်ပြီး လဲလှယ်သွားတဲ့ logic \n
Selection Sort: အသေးဆုံးဒေတာကို ရှာပြီး အရှေ့ကို ပို့ပေးတဲ့ နည်းလမ်း။
Insertion Sort: ဒေတာအသစ်ကို သင့်တော်တဲ့နေရာထဲသို့ ကြားညှပ်ထည့်သွင်းတဲ့ logic။
Shell Sort: Insertion sort ကို ပိုမိုမြန်ဆန်အောင် Gap (ကြားအကွာအဝေး) ခြားပြီး စီတဲ့နည်းလမ်း။
Quick Sort ⭐️: Pivot တစ်ခုထားပြီး အုပ်စုခွဲကာ အမြန်ဆုံး စီနိုင်တဲ့ logic။
Merge Sort ⭐️: Divide and Conquer နည်းစနစ်နဲ့ ဒေတာတွေကို ခွဲထုတ်ပြီး ပြန်ပေါင်းကာ စီတဲ့ logic။

---

### 🔗 Linear Data Structure
Linked List: Memory ထဲမှာ ဒေတာတွေကို တစ်ခုနဲ့တစ်ခု Pointer တွေနဲ့ ချိတ်ဆက်ပြီး dynamic ဖြစ်ဖြစ် သိမ်းဆည်းတဲ့ logic။
