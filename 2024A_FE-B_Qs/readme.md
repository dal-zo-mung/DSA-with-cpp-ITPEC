  **၁။ စာလုံးတစ်ခုချင်းစီရဲ့ အဓိပ္ပာယ်**
  
  pa (Stay Cycling): အရင်နှစ်က စက်ဘီးစီးတဲ့သူက ဒီနှစ်မှာလည်း ဆက်ပြီးစီးနေမယ့် နှုန်း။<br/>
  -> _တွက်ပုံ: $1 - pc = 1 - 0.3 = 0.7$ (70%)_ <br/>
  pc  (Stop Cycling): အရင်နှစ်က စက်ဘီးစီးတဲ့သူက ဒီနှစ်မှာ မစီးတော့ဘဲ ရပ်လိုက်မယ့် နှုန်း။<br/>
  ->  _ပေးထားချက်: $0.3$ (30%)_ <br/>
  pb (Start Cycling): အရင်နှစ်က မစီးတဲ့သူက ဒီနှစ်မှာ အသစ်စပြီး စီးလာမယ့် နှုန်း။<br/>
  ->  _ပေးထားချက်: $0.02$ (2%)_ <br/>
  pd (Stay Non-cycling): အရင်နှစ်က မစီးတဲ့သူက ဒီနှစ်မှာလည်း ဆက်ပြီး မစီးဘဲနေမယ့် နှုန်း။<br/>
  ->  _တွက်ပုံ: $1 - pb = 1 - 0.02 = 0.98$ (98%)_


**ပုစ္ဆာထဲက Matrix P မှာဆိုရင်:**

P[1,1] = pa (Row 1, Col 1) -> စီးရာမှ စီးခြင်း

P[1,2] = pb (Row 1, Col 2) -> မစီးရာမှ စီးခြင်း

P[2,1] = pc (Row 2, Col 1) -> စီးရာမှ မစီးခြင်း

P[2,2] = pd (Row 2, Col 2) -> မစီးရာမှ မစီးခြင်း
