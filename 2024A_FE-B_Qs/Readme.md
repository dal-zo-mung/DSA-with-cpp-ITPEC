**Q7: Longest Common Subsequence (LCS)**

ဒီပုစ္ဆာကတော့ Recursion (မိမိကိုယ်ကို ပြန်ခေါ်ခြင်း) ကို သုံးပြီး ဖြေရှင်းရတဲ့ နာမည်ကြီး Dynamic Programming ပုစ္ဆာတစ်ခု ဖြစ်ပါတယ်။ <br/>
str1 နဲ့ str2 ထဲက တူညီတဲ့ စာလုံးတွေကို လိုက်ရှာတာပါ။

Code ရဲ့ if (the m-th character... = the n-th character...) မှာ စာလုံးချင်းတူရင် 1 + lcs(..., m-1, n-1) ဆိုပြီး ရှေ့ကို ဆက်သွားပါတယ်။

စာလုံးချင်း မတူရင် (else) ဘာလုပ်မလဲ?
စာလုံးချင်း မတူရင် လမ်းခွဲ (၂) ခု ထွက်သွားပါတယ်။

(၁) ပထမစာကြောင်း (str1) က စာလုံးကို ကျော်ပြီး ဒုတိယစာကြောင်း (str2) နဲ့ ပြန်တိုက်ကြည့်မယ် -> lcs(str1, str2, m-1, n)
(၂) ဒုတိယစာကြောင်း (str2) က စာလုံးကို ကျော်ပြီး ပထမစာကြောင်း (str1) နဲ့ ပြန်တိုက်ကြည့်မယ် -> lcs(str1, str2, m, n-1)

_အဲဒီ (၁) နဲ့ (၂) ထဲက အများဆုံး (max) ထွက်လာမယ့် အဖြေကို ယူရမှာပါ။_

Code ထဲမှာ max ရဲ့ ပထမအပိုင်းက lcs(str1, str2, m, A) လို့ m ကို အသေထားတဲ့အတွက် n ကို လျှော့ရပါမယ်။ 

**ဒါကြောင့် A = n - 1 ပါ။**

ဒုတိယအပိုင်းက lcs(str1, str2, m - 1, B) လို့ m ကို လျှော့ထားတဲ့အတွက် n ကို အသေထားရပါမယ်။ 

**ဒါကြောင့် B = n ပါ။**
